## Objective
To measure SOC effectiveness using standard security metrics and communicate findings to both technical and non-technical stakeholders.

- Dwell Time

## Data Source
Metrics were derived from:
- Elastic Security alert timestamps
- TheHive case creation and status updates
- Simulated phishing incident timeline from previous tasks


## Reporting Levels
- Analyst-level: metrics calculation and interpretation
- Executive-level: high-level summary with recommendations
